  • Twitterと位置情報を組み合わせたようなwebアプリの開発

    • ＜PHP　HTML　SQL＞

    • ＜Python＞

    行列とベクトルの積、線形変換の考え方を用いて、曲名を全てアルファベット表記に統一し、どのアルバムにどのアルファベットがどの程度入っているかという行列を作り、曲名を入れることでどのアルバムと最も距離が近いか（入っている可能性が高いか）を示している。

    yolo5を用いて数種類の花を学習させ、複数の花が入った画像を入れるとそれを判別するシステムを作りました。
